# Massachusetts Housing Affordability Analysis

A county-level housing affordability analysis using official ACS Census API data for Massachusetts.

This project examines housing affordability across Massachusetts counties by comparing household income, home values, rents, cost-burden indicators, and estimated ownership costs under mortgage assumptions.

## Project Status

Initial analysis complete.

## Business Question

How affordable is housing across Massachusetts counties when comparing local incomes, home values, rents, estimated ownership costs, and cost-burden indicators?

## Current Scope

- Massachusetts county-level analysis.
- ACS 5-Year Census API data.
- Median household income.
- Median home value.
- Median gross rent.
- Owner/renter occupancy.
- Vacancy rate.
- Owner and renter cost-burden metrics.
- Estimated monthly ownership cost.
- Mortgage-rate sensitivity analysis.

## Tools

- Python
- pandas
- matplotlib
- Census ACS API

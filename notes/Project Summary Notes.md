# Massachusetts Housing Affordability Analysis — Project Notes



### Working Question

How affordable is housing across Massachusetts counties when comparing local incomes, home values, rents, estimated ownership costs, and cost-burden indicators?





### Current Scope



* Massachusetts county-level analysis.
* ACS 5-Year Census API data.
* Income, rent, home value, tenure, vacancy, owner/renter cost burden.
* Estimated monthly ownership cost using mortgage assumptions.
* Mortgage-rate sensitivity analysis.





### Current Outputs

* County affordability analysis dataset.
* Mortgage assumptions dataset.
* Mortgage-rate sensitivity dataset.
* County-level affordability charts.





### Possible Next Enhancements



* Add HUD Fair Market Rent data.
* Add HUD CHAS cost-burden data.
* Add MSA-level comparison.
* Add municipal or tract-level analysis.
* Add county map visualizations.
* Build a Tableau, Power BI, or Streamlit dashboard.
